# Car Comparison Cards

A simple project with three colored cards that shows different types of cars - Sedans, SUVs, and Luxury vehicles. Each card has a car icon, some description text, and a button that bounces when you hover over it.

## What You will find

- Three nice-looking cards side by side
- Interactive buttons that move up when you hover
- Clean, modern design with Poppins font
- Easy to customize colors and text

## Files You Need

index.html (the structure)
styles.css (the design)
asserts/ (folder with car icon images)

## How to Use It

Just clone the repository and open the `index.html` file in your browser. you will be shown the card;s structure and design.

If the icons don't show up, make sure your `asserts` folder is in the same place as your HTML file and has the car SVG images in them.

## Want to Change Something?

**Change card colors:**
Find `.card-1`, `.card-2`, or `.card-3` in the CSS and change the `background-color`.

**Make buttons bounce higher:**
Look for `transform: translateY(-8px)` and change the `-8px` to a bigger number like `-15px`.

**Change button colors:**
Find the button IDs (`#boss-1`, `#boss-2`, `#boss-3`) and change the `color` value or any other element you want to style.
